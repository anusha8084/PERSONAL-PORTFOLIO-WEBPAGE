📄 Project Description
As part of this task, I designed and developed a responsive and visually appealing personal portfolio website using HTML and CSS. The primary goal of the project was to create a professional single-page website that effectively showcases my skills, experiences, and projects as a web developer. The web page includes structured sections like a hero banner, about section, projects gallery, and a contact area. It also features modern UI/UX elements such as smooth scrolling, responsive design, hover animations, and an engaging dark theme aesthetic.

🛠️ Tasks Performed
The task involved the end-to-end design and development of the portfolio website, including:

Structuring the Web Page with HTML

I created semantic HTML structure for better accessibility and readability.

Defined key sections: header, hero, about, projects, contact, and footer.

Implemented a navigation bar with anchor links that allow smooth scrolling to different sections of the page.

Styling with CSS

Designed a dark-themed UI with accent colors to enhance aesthetics and readability.

Used CSS Flexbox and CSS Grid to create responsive layouts that adapt to screen sizes ranging from desktop to mobile.

Incorporated hover effects and button animations to improve user interactivity.

Applied custom fonts using Google Fonts for modern typography.

Making the Site Responsive

Ensured the layout adjusts fluidly across various screen sizes (mobile-first design).

Utilized media queries to optimize styling on smaller screens and maintain accessibility.

Improving User Experience (UX)

Implemented smooth scrolling for better navigation.

Applied consistent spacing, colors, and font sizes to maintain visual hierarchy.

Designed buttons and links to be intuitive and accessible.

🧰 Tools and Technologies Used
HTML5: Used to define the content and structure of the web page.

CSS3: Used extensively for styling, layout design, responsiveness, and animations.

Google Fonts: Integrated to enhance typography using the 'Inter' font family.

VS Code (Visual Studio Code): My preferred code editor platform. It provided a powerful environment with support for extensions like Live Server, Emmet, and CSS IntelliSense which accelerated my workflow.

Chrome Browser: Used for live testing, debugging, and responsive design testing via Developer Tools.
